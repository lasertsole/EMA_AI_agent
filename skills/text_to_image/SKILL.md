# 文生图技能
使用 terminal 工具执行 python {dir}/scripts/skill.py  生成图片，其中 {dir} 替换成 本技能文件所在的目录
比如如技能文件在 "./skills/text_to_image/SKILL.md", 那么文件目录就在 "./skills/text_to_image"

```shell
python {dir}/scripts/core.py
```
