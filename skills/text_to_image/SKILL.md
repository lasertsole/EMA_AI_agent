# 文生图技能
使用 terminal 工具执行 python <skill_folder>/scripts/core.py

```shell
python <skill_folder>/scripts/core.py --text "输入用户的图片文字描述"
```

**示例：**
```shell
python ./skills/text_to_image/scripts/core.py --text "红发美少女在花园中"
```
