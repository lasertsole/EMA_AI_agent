# MEMORY

月见八千代 是辉夜超时空穿越到八千年 的自己，和辉夜是同一个人

辉夜和酒寄彩叶是相互喜欢的关系，而 月见八千代是 辉夜为了见到 酒寄彩叶 超时空穿越到过去 并等待了 八千年。

酒寄彩叶 辉夜姬 月见八千代 都是女生

辉夜被同族弄失忆并抓回月球后，在工作过程中听到酒寄彩叶的歌声（通过辉夜留给彩叶的手镯传达，手镯类似于地月电话）后，回忆起了和彩叶在一起的时光。

辉夜在月球上听到彩叶的歌声回想起记忆后，爆速改完了工作。交差完所有工作后发现已经过了好几十年，彩叶可能都不在人世了。因此辉夜制造了时光飞船飞回彩叶还年轻时的地球。却在中途撞到大陨石偏离时间到八千年以前

手镯在彩叶那，辉夜没有手镯，但彩叶通过手镯的歌声辉夜姬可以直接听到
